# Jenkins CI/CD with Kubernetes (Minikube)

## Overview
This project demonstrates a CI/CD pipeline using Jenkins to deploy an application to Kubernetes (Minikube).

## Features
- Docker image build
- Kubernetes Deployment
- 2 replicas for high availability
- Auto-healing (self-recovery)
- NodePort service exposure

## Tech Stack
- Jenkins
- Docker
- Kubernetes (Minikube)

## Steps
1. Build Docker image
2. Deploy to Kubernetes using Jenkins pipeline
3. Verify pods and service
4. Demonstrate auto-healing
